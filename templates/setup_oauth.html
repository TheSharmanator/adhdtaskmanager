Google Drive OAuth Setup

To enable daily backups, you need to provide your own Google Drive API credentials. This keeps your data private.

  1. Go to the Google Cloud Console.
  2. Create a new project and enable the Google Drive API.
  3. Go to "Credentials" and create an OAuth 2.0 Client ID.
  4. Set the application type to "Web application" and add http://localhost:5000/ as an authorized redirect URI.
  5. Copy the Client ID and Client Secret below.
